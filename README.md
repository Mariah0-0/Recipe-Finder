# Recipe Finder

A React TypeScript application for discovering recipes using [TheMealDB](https://www.themealdb.com/) API.

## Install & Run

1. Clone the repository

```bash
git clone <your-repo-url>
cd recipe-finder
```

2. Install dependencies

```bash
npm install
```

3. Run

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS
- Vite
